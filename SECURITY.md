# Security Policy

## Supported Versions

Проект в активной разработке. Актуальная версия — текущая ветка `master`.

| Версия | Поддержка |
| ------ | --------- |
| master | :white_check_mark: |

## Reporting a Vulnerability

Если вы обнаружили уязвимость:

1. **Создайте Issue** в этом репозитории с пометкой `security`.
2. Опишите проблему и шаги воспроизведения.
3. Ответ ожидайте в течение 1–2 недель.
4. При подтверждении уязвимости — исправление в ближайшем обновлении.

Не публикуйте детали уязвимости до её исправления.
